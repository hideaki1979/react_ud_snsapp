# 【Udemu講座】簡易SNSアプリ（DBはsupabaseを使用）

## 概要
※本アプリはUdemy講座で作成したアプリです。
アカウント作成・ログイン機能とSNSの投稿機能・一覧表示機能があるReactで作成した簡易SNSアプリです。
